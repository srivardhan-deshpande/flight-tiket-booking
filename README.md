# Flight Ticket Booking Website

## Overview
This is a responsive Flight Ticket Booking web application built using HTML, CSS, and JavaScript. Users can search for flights by entering departure and destination cities, travel dates, and passenger details.

## Features
- Flight search functionality
- Departure and destination city selection
- Date selection
- Passenger selection
- Responsive design for desktop and mobile
- Modern and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in your browser.
